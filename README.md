# PySnake

Instructions:

1. You need Python 3
2. To run this you need also:
  
   Django==1.10.2
   
   djangorestframework==3.5.1  

   pygame==1.9.2b6
    
   requests==2.11.1
3. Run server first:
      python manage.py runserver
4. Run the game:
      python Main.py


PS. Pygame has problems with catching keystrokes when running in virtualenv
